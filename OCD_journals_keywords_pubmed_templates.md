# OCD Journals, Topic Positioning, Keywords, and PubMed Search Templates

**Scope:** This file is a **broad English reference list** for **obsessive-compulsive disorder (OCD)** research across **medicine, psychology, and neuroscience**.  
It includes:
1. **Direct OCD / OCRD-focused journals**
2. **High-yield psychiatry journals**
3. **High-yield clinical psychology / CBT journals**
4. **High-yield neuroscience / neuropsychiatry journals**
5. **Reusable PubMed search templates**

> Note: There is only a small number of journals that are explicitly OCD/OCRD-dedicated. Most OCD papers are published across broader psychiatry, psychology, psychotherapy, child psychiatry, neuropsychiatry, and neuroscience journals.

## 9) Practical Note

For OCD literature, the most efficient workflow is usually:

1. Start with the **broad OCD query**
2. Add one axis:
   - treatment
   - pediatric
   - neuroimaging
   - biomarkers
   - suicidality
   - family accommodation
   - treatment resistance
3. Add a **journal filter** only if you want a targeted shortlist
